# ParkVision 前端演示系统

这是 ParkVision 智慧停车综合管理平台的前端演示原型，所有文件都在当前项目目录内，不依赖联网安装。

## 如何运行

直接用浏览器打开：

```text
parkvision-frontend/index.html
```

## 已实现页面

- 总览驾驶舱：核心 KPI、车流预测图、库区概览、实时事件。
- 车主端 H5：查车位、预约、取车、临时取物、缴费、智能助手。
- 运营后台：出入场记录、计费规则、异常告警、黑白名单、自然语言报表。
- 数字孪生：车位网格、AGV 实时移动、安全急停联动。
- AI 感知：车牌识别、车位占用、人员闯入检测的结构化 JSON 演示。
- 调度中枢：优先级队列、VIP 插队、Pre-Dispatch 提前移库策略。
- 答辩脚本：按上一届系统实现模板整理汇报顺序。

## 演示建议

1. 点击“模拟入场”，展示 AI 识别车牌后创建订单、分配车位、事件流更新。
2. 进入“车主端 H5”，演示预约车位、我要取车、临时取物、缴费离场。
3. 进入“数字孪生”，展示 AGV 移动、库位状态和实时沙盘。
4. 点击“预调度”，展示高峰预测触发深层车辆迁移到浅层缓存区。
5. 进入“AI 感知”，点击“运行识别”，展示结构化 JSON 输出。
6. 点击“急停”，展示人员闯入时安全网关冻结 AGV 队列。

## 后续接入方向

- 将 `app.js` 中的模拟数据替换为 Spring Boot REST API。
- 数字孪生沙盘可接 WebSocket：`/ws/twin`。
- AI 识别结果可接 FastAPI：`POST /edge/vision/infer`。
- 车流预测结果可由 Prophet / LSTM 服务周期性写入后端，再推送给前端。
